# Microcontroller vs Microprocessor

| Parameter             | Microcontroller (Arduino UNO)                           | Microprocessor (Raspberry Pi)                                              |
| --------------------- | ------------------------------------------------------- | -------------------------------------------------------------------------- |
| **CPU Speed**         | 16 MHz                                                  | 1.5 GHz (model dependent)                                                  |
| **On-chip RAM**       | 2 KB SRAM                                               | No significant on-chip RAM; uses external RAM (typically 1 GB or more)     |
| **On-chip Flash**     | 32 KB Flash                                             | No on-chip Flash; boots from a microSD card                                |
| **Operating System**  | No operating system (runs a single program directly)    | Runs an operating system such as Raspberry Pi OS (Linux)                   |
| **Typical Use Case**  | Embedded systems, sensors, robotics, simple IoT devices | Multimedia, networking, edge computing, desktop applications, advanced IoT |
| **Power Consumption** | Low (typically less than 1 W)                           | Higher (typically 3–7 W depending on the model and workload)               |
